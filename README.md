# 1C:Наработки
Служат потенциальным работодателям для ознакомления с изученным и освоенным мной материалом в мире 1С.
Описание файлов:
1) Автосервис.dt и ПЗ.dt - Конфигурации разработанные на практических занятиях в период обученя. Максимально отржают опробованный функционал.
2) ИАР Степин.dt - Выпускная конфигурация. Приём заявок от пользователей, назначение сотрудникам, отслеживание. В основе работа с API телеграма.
3) УдалениеБитыхСсылокСФВ1.epf - Обработка для поиска битых ссылок в документах основаниях счет-фактур выданных, отображение списком, удаление. БП от ноября работало точно.
4) СР.cfe - потуги в сторону расширений. Доработка печатной формы. Функция ВКЛ/ВЫКЛ изменений в печатной форме инвентаризации по галочке в специальной форме настроек расширения.
