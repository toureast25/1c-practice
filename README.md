# 1C:Наработки
Служат потенциальным работодателям для ознакомления с изученным и освоенным мной материалом в мире 1С.
Описание файлов:
1) Автосервис.dt и ПЗ.dt - Конфигурации разработанные на практических занятиях в период обученя. Максимально отржают опробованный функционал.
2) ИАР Степин.dt - Выпускная конфигурация. Приём заявок от пользователей, назначение сотрудникам, отслеживание. В основе работа с API телеграма.
3) УдалениеБитыхСсылокСФВ1.epf - Обработка для поиска битых ссылок в документах основаниях счет-фактур выданных, отображение списком, удаление. БП от ноября работало точно.
4) СР.cfe - потуги в сторону расширений. Доработка печатной формы. Функция ВКЛ/ВЫКЛ изменений в печатной форме инвентаризации по галочке в специальной форме настроек расширения. 
5) ВО_Степин_1.erf
    Мой первый отчёт, построенный по СКД.
    Освоено:
    1. Настройка запроса к таблице ОстаткиИОбороты регистра накопления.
    2. Описание полей и ролей
    Реализовал корректный расчет итогов по периодам. Настроил роли «Начальный остаток» и «Конечный остаток» с привязкой к полю периода, чтобы избежать некорректного суммирования остатков в группировках.
    3. Работа с ресурсами
    4. Настройка структуры и интерфейса
    Иерархические группировки: Создал многоуровневую структуру (Организация -> Подразделение -> Сотрудник).
    Выбранные поля: Сформировал состав колонок отчета, разделяя показатели на входящие остатки, обороты и конечный результат.
    Параметризация: Настроил стандартные параметры периода для гибкой фильтрации данных пользователем.
